# Ontologist Workspace - MoneyWorks Canonical Semantic Ontology Project

## Overview
This workspace coordinates ont1 and ont2 ontologists in reconstructing MoneyWorks canonical semantic ontology to achieve 100% compliance with our established methodology.

## Workspace Structure
```
./generated/ontologist-workspace/
├── README.md                    # This file
├── MASTER-INSTRUCTIONS.md       # Core methodology and standards
├── PRIORITY-ASSIGNMENTS.md      # Work assignments and priorities
├── ont1/                        # ont1 workspace
│   ├── assignments/             # Task assignments for ont1
│   ├── reports/                 # Completion reports from ont1
│   └── outputs/                 # Ontology files created by ont1
├── ont2/                        # ont2 workspace
│   ├── assignments/             # Task assignments for ont2
│   ├── reports/                 # Completion reports from ont2
│   └── outputs/                 # Ontology files created by ont2
└── coordination/                # Cross-ontologist coordination
    ├── quality-reviews/         # Quality assessment reports
    ├── integration-tests/       # Cross-entity validation tests
    └── progress-tracking/       # Project status tracking
```

## Ontologist Workflow
1. **Assignment Phase**: Receive specific entity extraction assignments
2. **Execution Phase**: Follow `/extract-moneyworks-entity` methodology exactly
3. **Reporting Phase**: Submit completion reports with validation results
4. **Review Phase**: Undergo quality assessment and integration testing

## Quality Standards
- 100% field coverage from manual sources
- Complete foreign key relationship documentation
- Modern field object structure compliance
- Comprehensive validation test suites
- Cross-business universality validation

Each entity must achieve "Names-level excellence" (95%+ coverage, 100% relationships, architectural insights)
