Adding Reminder Messages

  1. Choose Show>Reminder Messages

The Reminder Messages list is displayed.

  1. Choose Edit>New Message or press Ctrl-N/⌘-N to add a new message, or double click an existing one to change it

The Reminder Message dialog box opens.

  1. Enter or change the message frequency

The Reminder Message specification box works in the same way as the Define Recurring Transaction box— see Recurring Transactions. Use the Once Only option if you only want the message to appear once on a specific date.

  1. Enter the text for the message into the Message box at the base of the dialog box
  1. Click OK to enter the message

The message will appear in the Reminder Messages window.
