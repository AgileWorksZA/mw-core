# Reports

## Overview
MoneyWorks includes a comprehensive reporting engine with built-in reports across all functional areas, plus a custom report editor for creating tailored reports.

## Access Points
- Right sidebar: Reports icon
- Menu bar: Reports menu
- Chart of Accounts navigator: Reports section
- Names list sidebar: Report on selection...
- Items list sidebar: Report on selection...
- Transaction list sidebar: Report on selection...

## Report on Selection
Many list views include a "Report on selection..." section in their sidebar. This runs the selected report against the currently filtered/selected records.

## Custom Reports

### Report Editor
- Edit menu → Report Editor → submenu
- Design custom report layouts
- Select data sources, fields, grouping, sorting
- Formula support
- Header/footer/detail band design
- Export to PDF, print, or screen

### Forms Designer
- Edit menu → Forms Designer → submenu
- Design custom forms (invoices, statements, POs)
- Used for printing customized transaction documents
- Template-based design

## Report Output Options
- **Screen**: Preview in window
- **Print**: Send to printer
- **PDF**: Export as PDF file
- **Export**: CSV or text file export

## Business Rules
- Reports use posted data by default
- Period filtering available on all financial reports
- Department filtering when departments are enabled
- Category filtering for categorized data
- "Report on selection" runs against the currently visible filtered list
