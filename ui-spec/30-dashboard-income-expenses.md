# Dashboard: Income and Expenses

## Overview
Shows income and expenses broken down by category for a selected period. Provides detailed P&L analysis at a glance.

## Access Points
- Navigator: Dashboard → Income and Expenses

## Layout
Title: "Income and Expenses by Category" with company name and period selector (e.g., "Period Jan:2024/25" with navigation arrows)

### Content
- Displays income accounts and expense accounts grouped by category
- Shows actual amounts vs budget (when budgets are set)
- Empty state message: "These accounts contain no posted transactions for this year. Please post transactions to see results in the Dashboard"

## Features
- Period selector with forward/back arrows
- Category-based grouping
- Budget comparison (when available)
- Requires posted transactions to display data
