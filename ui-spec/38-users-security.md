# Users and Security

## Overview
Manages user accounts, roles, and permissions for multi-user access to MoneyWorks files.

## Access Points
- File menu: Users & Security...
- File menu: Switch User...
- File menu: Signing...

## Features

### User Management
- Create/edit/delete user accounts
- Set passwords
- Assign roles/permission levels
- Enable/disable users

### Permission Levels
- Full access
- Read-only
- Department-restricted
- Transaction-type restrictions
- Account-level security (Security Level on accounts)

### Signing
- Digital signing for transaction approval
- Audit trail of who created/modified records
- Approval workflows

### Switch User
- Quick user switch without closing the file
- Different users see different data based on permissions

## Security Model
- File-level authentication (users are per-document)
- Account Security Level field restricts who can post to sensitive accounts
- Transaction-level audit trail (who created, who modified, timestamps)
