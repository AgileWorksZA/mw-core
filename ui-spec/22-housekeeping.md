# Housekeeping

## Overview
Housekeeping is the system administration hub. It shows company information, file statistics, data management tools, and report/form customization. Think of it as the "settings and tools" dashboard.

## Access Points
- Navigator: Housekeeping

## Layout — Three Sections

### Company Details Section
- Displays company name, postal address, delivery address
- Shows company logo image
- Quick access buttons:
  - **Users & Security** — User/permission management
  - **Document Preferences** — Per-file settings (9 tabs of configuration)
  - **MoneyWorks Preferences** — Application-wide settings
  - **Check for Software Update** — Shows current version (e.g., "You are using version 9.2.3")

Quick access buttons:
- **Log File** — View system log
- **Diagnostics** — Database integrity checks

### Reports and Forms Section
| Action | Description |
|--------|-------------|
| **Index to Reports** | Browse all available reports |
| **New Custom Report** | Create a custom report from scratch |
| **New Analysis Report** | Create an analysis/pivot report |
| **New Report Chain** | Chain multiple reports together |
| **New Custom Form** | Design a custom form (invoice, PO, etc.) |
| **Sign and share Reports and Forms** | Share custom reports/forms with others |

Footer: "Set navigator background image" / "Remove navigator background image" links
