If you are heavily into budgets, you will be pleased to know that MoneyWorks
supports two alternate budgets, A and B. Budget information for each of these
can be entered for the Previous, Current and Next financial years, and is used to
generate variance data in your account enquiries and reports. Budgets are
normally done on an accrual basis (e.g. invoiced sales); use the Cash Forecast
report to convert an accrual budget to a 15 month cash flow forecast.

Budget data can be entered directly into MoneyWorks, it can be copied out of a
spreadsheet (or from the MoneyWorks historic actual or budget data) and
pasted into the MoneyWorks budget editor, or it can be imported from a text
file (import).

Reminder Messages

The day’s messages (and any messages due since you last opened
MoneyWorks) will be displayed when you open your MoneyWorks
document. Use the Today’s Messagescommand in the Show menu to view
them again at any time during a work session.

You can click the Keep check box to bring up the message on subsequent days.
Uncheck the Keep box to stop the message from appearing until it is next due.

System Messages

In addition to reminder messages you have put in, the following is displayed:

If there are any Name records that are due to follow up, a single message

will be displayed with a telephone icon next to it. Double clicking on this
reminder will display a list of the contacts needing follow-up.

If any recurring transactions have been generated (or should have been

generated), there will be a message to that effect.

If the Overdue Warnings at StartupDocument preference item is on, a
notification of any overdue creditor or debtor invoices. Double clicking
these warnings will display the recalcitrant transactions in the Payables and
Receivables list respectively.

Use the left and right arrow keys to view the messages for (respectively) the
previous and next day. When the message has served its purpose, it can be
deleted.
