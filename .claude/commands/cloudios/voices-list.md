#!/usr/bin/env bun run ${PWD}/.claude/commands/cloudios/voices-list.ts

# List available TTS voices

Shows all available text-to-speech voices in CLOUDIOS.

Usage: `/cloudios:voices-list [provider]`

Examples:
- `/cloudios:voices-list` - List all voices
- `/cloudios:voices-list elevenlabs` - List only ElevenLabs voices
