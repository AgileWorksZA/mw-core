#!/usr/bin/env bun run ${PWD}/.claude/commands/jarvis/wip.ts

# Pick up the next work item

Finds and assigns the next kanban task to work on.

Usage: 
- `/jarvis-wip <agent-name>` - Pick up tasks assigned to you
- `/jarvis-wip <agent-name> next` - Pick up any available task

Example: `/jarvis-wip herman-jarvis`
