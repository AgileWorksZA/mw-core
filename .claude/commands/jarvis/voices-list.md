# List available TTS voices

Shows all available text-to-speech voices in JARVIS.

Usage: `/jarvis:voices-list [provider]`

Examples:
- `/jarvis:voices-list` - List all voices
- `/jarvis:voices-list elevenlabs` - List only ElevenLabs voices
